HPE Storage FlowKit

Python SDK & Orchestrated Workflow Library for HPE Storage REST APIs

API Reference Source Repository

Overview

HPE Storage FlowKit is a Python workflow-centric toolkit that simplifies interacting with HPE Storage arrays. It layers orchestration and idempotent behaviors over the raw REST API endpoints for objects such as volumes, snapshots, hosts, host sets, volume sets, VLUN exports, QoS policies, CPGs, and remote copy groups.

Get Started: Use the AnsibleClient composite class or individual workflow classes for direct control. See API Reference for detailed method semantics and REST mappings.

Composite Usage

The AnsibleClient class inherits all workflows, exposing a unified surface area for automation frameworks. Each call performs minimal preprocessing then delegates to a workflow method that directly maps to a REST endpoint.
