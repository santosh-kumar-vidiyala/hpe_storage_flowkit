RemoteCopyGroupWorkflow

Remote copy group replication operations.

API Reference Home

Class Summary

The RemoteCopyGroupWorkflow class manages asynchronous/synchronous replication groups, including lifecycle (start/stop/sync), volume admission/removal, and target/link management (REST + SSH CLI commands).
